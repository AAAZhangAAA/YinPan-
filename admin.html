☁ 直接保存到 GitHub（免手动上传）

      

        填一次 GitHub 访问令牌（需 repo 权限），之后点「保存到 GitHub」即自动把内容提交到仓库，
        GitHub Pages 会自动重建，粉丝约 1–2 分钟后就能看到更新——你无需再改文件、无需手动上传。

        ⚠️ 令牌仅保存在你当前浏览器（localStorage），且有仓库写入权限，请勿在公共电脑上使用。
